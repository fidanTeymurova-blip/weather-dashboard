# Weather Dashboard 🌦

Responsive weather app built with React + TypeScript using OpenWeatherMap API.

## Features
- Real-time weather data
- 5-day forecast
- City search
- Dark / Light mode

## Tech Stack
- React
- TypeScript
- OpenWeatherMap API
- Vite
